# **App Name**: EduQuiz MVP

## Core Features:

- Quiz Headboard: Progress tracker showing the current question number relative to the total set (e.g., 'Soal 1 dari 3').
- Question Portal: A focused container for the question text ensuring maximum readability and whitespace.
- Vertical Option Matrix: Four stacked interactive selectors for answers A through D, designed for touch-friendly accessibility.
- Sequential Navigator: A persistent footer button that handles state transitions and moves the user to the next question automatically.
- AI Explanation Tool: A tool that uses generative AI to provide a simple, context-aware explanation for why an answer is correct after the quiz completes.
- Local Mock Assessment: A self-contained state-management system to handle quiz data locally without external latency.
- Performance Summary: A final scorecard displayed after the last question is answered to summarize user performance.

## Style Guidelines:

- Primary color: Deep Intellectual Blue (#1F52AD) to evoke focus and academic reliability.
- Background color: Clean Paper White (#F6F7F9), a desaturated variant of the primary hue to reduce eye strain.
- Accent color: Insight Teal (#237E8C) used for highlighted selection states and supplementary icons.
- Font recommendation: 'Inter' sans-serif for its machining clarity and professional readability in dense exam layouts.
- Minimalist centered layout with a vertical stacking principle to mimic physical examination forms.
- Standardized thin-line icons for navigational actions to maintain a lightweight UI feel.
- Subtle horizontal slide transitions when moving between question states to indicate linear progress.
